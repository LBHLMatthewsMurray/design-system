Typography

For GOV.UK domains, always use GDS Transport webfont in Light and Bold. For internal services, use Arial.

Type sizes

Body

  • we recommend starting with 19px for body content
  • 16px should be the minimum size of body content

A 19px body of content, a patterns that has been developed by the Home Office DDaT user research and design teams to help create consistent, usable services.

A 16px body of content, which should be the minimum type size so users are able to read content comfortably.

Headings

  • follow the sizes and guidance for GOK.UK services
  • use sentence case for headings
  • use headings consistently to create a clear hierarchy

A 48px Bold heading

A 36px Bold heading

A 24px Bold heading

A 19px Bold heading

Line heights

We recommend a line height of 1.3 to acheive easy to read content.

  • the root line height for 19px body content is 25px
  • the root line height for 16px body content is 21px
  • paragraphs should have a margin-bottom equal to the body text line height

Line lengths

The length of a line of content is an important factor in how easy it is to read. Line lengths should be between 50 - 75 characters.
