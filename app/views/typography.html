Typography

For GOV.UK domains, always use GDS Transport webfont in Light and Bold.
For internal services, use a Helvetica/Arial font stack.

Font

A Helvetica/Arial font stack already exists in the frontend/prototype toolkit. Use Sass variable $helvetica-regular which declares "HelveticaNeue", "Helvetica Neue", "Arial", "Helvetica", sans-serif;
